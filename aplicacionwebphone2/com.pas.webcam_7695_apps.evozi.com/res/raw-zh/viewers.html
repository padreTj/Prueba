Android版tinyCam Monitor

设置tinyCam Monitor

Alexey Vasilyev的tinyCam Monitor是一款Android远程监控软件，支持所有主流摄像头厂家，包括IP摄像头。

专业版支持IP摄像头的音频和扩展功能，例如控制闪光灯和自动对焦。

tinyCam查看器和IP摄像头协同工作的视频 (需要互联网连接).

在此获取 或者使用扫码软件扫描该二维码

在软件中，选择“管理摄像头”，添加一个新的摄像头，在厂家列表中选择 Android IP网络摄像头（IP Webcam for Android）。

输入主机名 $IP$, 选择你已经配置的端口号 $PORT$， 然后就大功告成了。

VLC媒体播放器

VLC媒体播放器是一个具有跨平台的视频播放器、流式媒体服务器和转换器的集成解决方案。在此获取.

如果想要使用VLC媒体播放器和IP摄像头协同工作，在一VLC播放器的菜单中选择媒体⇨打开网络流媒体。输入$URL$/video来流式传输视频或$URL$/audio.wav流式传输音频。

你也可以使用VLC进行视频录制：

  • 选择 媒体 -> 转换/保存.
  • 选择 网络 选项卡
  • 在URL栏输入 $URL$/video
  • 点击 转换/保存
  • 选择文件保存路径和格式即可

    通过本地WiFi之外访问IP摄像头（例如通过互联网）

    你可以在手机连接到WiFi，并且路由器拥有外网IP时（大多数情况都会有的），通过互联网访问你的手机。配置需要一些工夫，不过大多数人都能应付。

    重要：请一并阅读 MAC 绑定指引 以防止内网IP变化。

    在你的路由器的网页管理界面，你应该可以将服务器端口转发到互联网上，并通过路由器的外网IP来访问摄像头。这个功能通常叫做“端口转发”，“虚拟服务器”。你可以参考路由器的说明书来学习如何配置。这里提供给你一个免费的端口转发指南http://portforward.com/。可以无视付费内容，你只需要指南。

    设置看起来像这样：

    Phone (192.168.3.198, server at 8080) ---> Router (Forwarding 192.168.3.198:8080, external IP is 66.77.88.99) <--- Any client with internet access (by opening http://66.77.88.99:8080).

    这里提供一个通过互联网确定你的外网IP，检查IP摄像头可见性的快捷方法。如果你在路由器设置中找不到外部IP，你可以访问http://whatismyip.org。如需检查你是否正确配置了你的路由器，使用任意在线端口扫描器，例如这个。将你的外网IP输入到IP栏中（例子里是66.77.88.99），并把外部转发端口填到“端口列表”框中（例子里是8080）。

    提醒 你可能不能通过路由器的外部界面访问你的手机，即使你和该手机处于同一网络中。这是一个常见的路由器固件错误。你可能需要进入其他WiFi网络或通过3G完成这个操作。

    为什么IP地址总是在变化？

    你的路由器负责为你分配IP。分配的IP有一定时限，超时后就会更换。如果是这样，可能导致无法从上一个地址访问IP摄像头。

    要防止这个问题发生，你需要进入路由器的网页管理界面，为你的设备分配一个静态IP。寻找菜单项中类似于“DHCP服务器”，“静态路由表”或“IP/MAC绑定”，然后在那里为MAC绑定你想要的、或者当前的IP。

    你的MAC地址可以在这里找到：设置->关于手机->状态信息->WLAN MAC 地址.
