¿Por qué oigo el sonido con 4 segundos de retraso?

Esto ocurre debido al búfer del navegador y depende de cuál que se use.

Intente resolverlo de la siguiente forma:

  • Use el renderizador Flash, que está diseñado para reducir el intervalo.
  • Use Firefox y audio Opus. Opus es un códec de audio de próxima generación con latencia y ancho de banda bajos, y Firefox usa menos el búfer.
  • Use VLC Media Player, y en el menú abra Media ⇨ Abrir Retransmisión de Red, introduzca http://your-ip:your-port/audio.wav como enlace, haga clic en "Mostrar más opciones" y cambie el valor del búfer a 1 ms.

Control de eventos Tasker

Tasker es una solución de automatización para los celulares Android. Con él puede enviar un SMS o cambiar el volumen de su tono de llamada, dependiendo de su localización o de la hora.

IP Webcam Pro puede generar eventos Tasker, tanto como con la interfaz web como con la personalizable.

Para configurar Tasker para usar eventos de IP Webcam, cree un nuevo Perfil usando el botón "+" y seleccione Estado ⇨ Plugin ⇨ IP Webcam como disparador.

Los eventos generados desde la interfaz web tienen el mismo nombre de evento que el texto del botón. Por ejemplo, el botón central genera un evento con el nombre "5".

Avisos cuando se utiliza HTTPS?

Es normal que aparezcan avisos sobre el HTTPS cuando no se accede al host usando DNS.

El utilizar HTTPS proteje de ataques cuando se utilizan redes inseguras.

Es seguro ignorar esos avisos.
