TinyCam Monitor per Android

Impostare tinyCam Monitor

tinyCam Monitor di Alexey Vasilyev è un'applicazione Android per la sorveglianza a distanza, supportando tutti i principali fornitori di telecamere, tra cui IP Webcam.

Supporta l'audio di IP Webcam (Versione Pro) e funzioni estese, come il controllo dei LED e autofocus.

Video che mostra Webcam IP e tinyCam Monitor in azione (connessione internet necessaria).

Vai qui o scannerizza con QR code Barcode scanner

Nell'applicazione, selezionare "Gestione camera", aggiungere una nuova fotocamera, selezionare vendor "IP Webcam per Android".

Inserite l’hostname $IP$, selezionate la porta configurata $PORT$ e siete a posto.

VLC Media Player

VLC Media Player è un lettore video atform cross-pl, streaming server convertitore in un unico pacchetto. Vai qui.

Per l'utilizzo di IP Webcam con VLC media player, seleziona nel menù Media ⇨ aperta Network Stream e immettere $URL$/video per lo streaming video o $URL$/audio.wav per lo streaming audio.

È inoltre possibile utilizzare VLC Media player per la registrazione video:

  • Seleziona Media -> Converti/Salva.
  • Seleziona scheda rete
  • Inserisci $URL$/video come URL
  • Clicca converti/salva
  • Selezionare il file di destinazione, il formato in cui si desidera salvare e siete pronti a partire

    Note: Se necessario, specificare il nome utente la password in URL, come questo: username: password@$IP$.

    Blue Iris

    Apri "Aggiungi camera" dialogo come al solito

    Quando si utilizza IP Webcam, devi scegliere "MJPEG Stream" dalla lista invece di una specifica telecamera.

    Percorso video:/video
    Percorso Audio:/audio.wav
    Nome_host: $IP$
    Porta:$PORT$

    Non è fondamentale che tu inserisca la spunta "RTSP port / video" .

    webcamXP

    Nell'interfaccia principale webcamXP, fate clic destro sulla sorgente video e selezionare Telecamere di rete ⇨ Connect ...

    Settare Camera Marca a Android e Modello in IP Webcam. Selezionare la voce desiderata (MJPEG consigliato). Clicca Avanti.

    Impostare Hostname per $IP$ e Porta per $PORT$.

    Fare clic su OK e si è pronti a utilizzare la nuova macchina fotografica!

    Avanzata

    Ecco l'elenco degli URL dei servizi IP Webcam:

    • $URL$/video è l'URL MJPEG.
    • $URL$/shot.jpg recupera l'ultimo fotogramma.
    • $URL$/audio.wav è il flusso audio in formato Wav.
    • $URL$/audio.aac è il flusso audio in formato AAC (se supportato dall'hardware).
    • $URL$/audio.opus è il flusso audio in formato Opus.
    • $URL$/focus concentra la fotocamera.
    • $URL$/nofocus rilascia la messa a fuoco.

    Accesso IP Webcam fuori WiFi locale (ad esempio via Internet)

    È possibile accedere al telefono via internet quando il telefono è connesso a una rete WiFi e il router WiFi ha un IP esterno. La configurazione richiede un po 'di lavoro, ma la maggior parte delle persone la riesce a gestire.

    Importante: Leggere anche la guida su come vincolare il mac per evitare che il vostro IP interno cambi.

    Tramite interfaccia web del router, è necessario inoltrare la porta del server a Internet e accedere alla videocamera tramite IP esterno del router. Questa funzione viene di solito chiamato "Port forwarding", "server virtuali". Dovreste vedere la documentazione del router sul come farlo. C'è anche una banca dati gratuita di manuali per il port forwarding su http://portforward.com/. Basta Evitare cose a pagamento, Avrai solo bisogno dei manuali.

    La configurazione sarà simile a questa:

    Telefono (192.168.3.198, server presso 8080) ---> Router (Inoltro 192.168.3.198:8080, IP esterno è 66.77.88.99) <---Qualsiasi client con accesso ad internet con l'apertura di http://66.77.88.99:8080).

    C'è una scorciatoia nel determinare il tuo IP esterno e controllare la IP Webcam su Internet. Se non riesci a trovare il tuo IP esterno nelle impostazioni del router, si può ottenere visitando http://whatismyip.org. Per controllare se avete configurato correttamente il router, utilizzare qualsiasi port scanner online, come questo. Inserisci il tuo IP esterno nel campo IP (nel mio esempio, 66.77.88.99), e la porta esterna trasmessa in "lista delle porte" campo (8080 nel mio esempio).

    Note probabilmente non sarete in grado di accedere al vostro telefono dall'interno della stessa rete del router. Questo è un bug frequente alla maggior parte dei router. Dovrete essere in un’altra rete WiFi o su reti 3G per controllare il corretto funzionamento del port-forwarding. Spesso i router non eseguono il redirezzionamento.

    Perché l’indirizzo IP continua a cambiare?

    Al dispositivo viene assegnato un IP dal vostro router WiFi. L'assegnazione ("lease") può durare un po di giorni e poi cambiare. Questo potrebbe rendere IP Webcam inaccessibile al precedente indirizzo ip.

    Per evitare che ciò accada, è necessario richiedere al proprio gestore un indirizzo IP statico ed impostare sul router il vincolo di collegamento IP/MAC. Ricerca nel menù voci simili a "client DHCP", "leasing statici" o "binding IP / MAC", e aggiungere un MAC vincolante qui all' IP corrente desiderato.

    L’indirizzo MAC del telefono può essere trovato in Impostazioni-> Info sul telefono indirizzo Status-> WiFi MAC.
