Android版tinyCam モニター

tinyCam モニターの設定方法

Alexey Vasilyevにより開発されたtinyCam Monitorは遠隔監視システム用のAndroidアプリケーションです。IP Webcamを含むほとんどのカメラアプリをサポートしています。

このアプリはIP Webcamの音声機能 (Pro版のみ)やLEDコントロール、オートフォーカスコントロールなどの拡張機能に対応しています。

映像では起動中のIP WebcamとtinyCam Monitorを確認できます。 (インターネット接続が必要です)

ダウンロードはこちらをクリックするか、バーコードリーダーを使ってQRコードをスキャンして下さい。

アプリケーションでは、 "Manage cams"を選択し、新しいカメラを追加し、"IP Webcam for Android"を選択します。

ホスト名に$IP$を入力し、 設定したポート$PORT$を設定すると利用できます。

VLC メディアプレーヤー

VLC Media Playerはクロスプラットフォームの動画プレイヤーです。このソフトは単独でストリーミングサーバーとしての機能やファイルの変換を行うことができます。ダウンロードはこちら。

VLC Media PlayerでIP Webcamを使用するためには、メニューから以下の設定を行って下さい。Media タブからNetwork Streamを開き、ストリーミングビデオ欄に$URL$/videoと入力するか、ストリーミング音声欄に$URL$/audio.wavを入力します。

ビデオ録画にもVLC Media Playerが使用できます。

  • Mediaタブから 変換/保存を選択します。
  • ネットワークタブを選択します。
  • URL欄に$URL$/videoを入力します。
  • 変換/保存をクリックします。
  • ファイルの保存先を選択し、ファイル形式を選ぶと準備完了です。

    ZoneMinder

    下記の通り設定してください。

    一般：
    ソースタイプ： 遠隔
    機能： モニター

    ソース：
    プロコトル： HTTP
    メソッド： シンプル
    ホスト名： $IP$
    ポート： $PORT$
    リモートホストパス： /video (videoだけでは起動しません)。

    注意： 必要であれば、username:password@$IP$のような形でURL欄にユーザー名とパスワードを明記してください。

    Blue Iris

    "Add camera" をクリックしてダイアログを開きます。

    IP Webcamを使用する際はリストから"MJPEG Stream"を選択して下さい。

    ビデオパス： /video
    音声パス： /audio.wav
    ホスト名： $IP$
    ポート： $PORT$

    「RTSP/video port」欄に入力した内容は設定に特に影響はありません。

    webcamXP

    webcamXPのメイン画面にあるビデオソース上で右クリックし、ネットワークカメラ ⇨ 接続を選択します。

    カメラ機種をAndroidに設定し、カメラモデルをIP Webcamに設定します。プリセットから映像形式を選択します。(MJPEG推奨) 次をクリックします。

    ホスト名に$IP$を、ポートに$PORT$を入力します。

    OKをクリックすると設定が完了し、カメラを使用することができます。

    詳細

    IP WebcamサービスURL一覧：

    • $URL$/videoはMJPEGのURLです。
    • $URL$/shot.jpgは最新フレームを取得します。
    • $URL$/audio.wavはWav形式で音声をストリーミングします。
    • $URL$/audio.aacはAAC形式で音声をストリーミングします。 (ハードウェアでサポートされている場合のみ)
    • $URL$/audio.opusはOpus形式で音声をストリーミングします。
    • $URL$/focusはカメラがフォーカスします。
    • $URL$/nofocusフォーカスを解除します。

    ローカルネットワーク外でのIP Webcamへのアクセス (インターネット経由など)

    お使いの端末がWiFiに接続していて、使用中のWiFiルーターがグローバルIPに対応している場合(ほとんどのルーターはグローバルIP機能を搭載しています)、インターネット経由でAndroid搭載端末にアクセスすることができます。この機能を利用する場合、簡単な設定変更を行う必要があります。

    重要：ローカルIPアドレスの変更を防ぐため、MACアドレス固定ガイドをお読み下さい。

    お使いのルーターのウェブ設定ページ上で、インターネットにサーバーポートを転送させルーターのグローバルIPを利用してカメラに接続します。この機能は「ポート転送」、「仮想サーバー」と呼ばれています。詳しい設定方法はルーターの説明書をご確認下さい。

    設定方法は以下のように行って下さい。

    電話 (ローカルIP：192.168.3.198  ポート：8080) ---> ルーター (ポート転送設定：192.168.3.198:8080　ルーターグローバルIP 66.77.88.99) <--- ネット接続したクライアントで次のようにアドレスを入力します。(http://66.77.88.99:8080)

    グローバルIPの設定とインターネット上でのIP Webcamの視聴確認は簡単に行うことができます。グローバルIPがわからない場合は、http://whatismyip.orgにアクセスすることでグローバルIPを確認することができます。お使いのルーターが正しく設定されているかどうか確認するには、こちらからご確認ください。（英語のみ）IPアドレス入力欄にグローバルIPを入力してください。(たとえば、66.77.88.99など) ポート入力欄にはポート番号を入力します。(たとえば、8080など)

    ルーターのグローバルIPを利用してアクセスしようとする場合に、同一ネットワーク内のルーターに接続しようとするとAndroid端末に接続できない場合がありますのでご注意ください。これはルーター側の設定で頻繁に起こるバグです。グローバルIPを利用してAndroid端末にアクセスする場合は、他のWiFiネットワークを利用するか3G回線を利用してください。

    なぜIPアドレスが変更してしまうのか。

    IPアドレスは、WiFiルーターにより各端末に割り振られています。IPアドレスの割り振り は一定期間は同一のIPアドレスが使用されますが、一定期間を過ぎると自動的にIPアドレスは別のアドレスに変更されます。これによりセキュリティ性を確保しています。（同じIPアドレスでIP Webcamにアクセスすることができなくなります）

    IPアドレスが常に変更されることを防ぐためには、ルーターの設定画面で端末に固定IPを割り振る必要があります。「DHCPクライアント」、「固定開放」 または 「IPアドレス/MACアドレス固定」というような名称の設定項目をメニューから探し、MACアドレスと希望のIPアドレスか現在のIPアドレスに固定する設定をします。

    Android端末のMACアドレスは設定->端末情報->ステータス->WiFi MACアドレスで確認することができます。
