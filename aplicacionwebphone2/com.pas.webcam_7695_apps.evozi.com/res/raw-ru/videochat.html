IP Camera Adapter

Мой драйвер не имеет баннера для бесплатной версии и свободен для любого использования.

Скачайте его отсюда.

После установки драйвера откройте окно конфигурации, введите $URL$/video в поле для URL, заполните имя пользователя и пароль (если они установлены) и нажмите Autodetect для автоматического определения разрешения.

Теперь можно запускать приложение для видеочата и пользоваться эмулируемой камерой!
