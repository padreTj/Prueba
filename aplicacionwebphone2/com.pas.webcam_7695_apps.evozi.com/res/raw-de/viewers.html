tinyCam Monitor for Android

Aufsetzen des tinyCam Monitor

tinyCam Monitor von Alexey Vasilyev ist eine Android Applikation für Beobachtung aus Entfernung, unterstützt von allen grossen Kamera Herstellern, inklusive IP Webcam.

Es unterstützt IP Webcam's audio (Pro version) und die zusätzlichen Features, wie steuern der LED und des Autofocus.

Video showing IP Webcam und tinyCam Monitor in action (internet connection erforderlich).

Get it here oder via diesen QR Code mit dem Barcode scanner

In der Applikation, selektieren Sie "Manage cams", add a new Kamera, und select vendor "IP Webcam for Android".

Tippen sie den hostname $IP$ ein, selektieren den konfigurerten Port $PORT$ und sie sind startbereit.

VLC Media Player

VLC Media Player ist ein platformübergreifender Video Player, Streaming Server und Converter Lösung in einem single package. Hier Bekommen.

Um IP Webcam mit dem VLC media player zu konfigurieren, im Menu selektieren sie Media ⇨ Open Network Stream und tippen sie $URL$/video

für streaming video oder $URL$/audio.wav für streaming audio.

Sie können auch den VLC Media player für Video Recording benutzen:

  • Selektieren Sie Media -> Convert/Save.
  • Selektieren Sie Network tab
  • Klicken sie $URL$/video als URL
  • Klicken sie convert/save
  • Selektieren Sie Destination File, formatieren sie und speichern sie das File ab

    ZoneMinder

    Benutze diese Konfiguration:

    Generell:
    Source type: Remote
    Function: Monitor

    Source:
    Protocol: HTTP
    Method: Simple
    Host name: $IP$
    Port: $PORT$
    Remote host path: /video (just "video" doesn't work).

    Note: Falls notwendig, spezifizieren Sie den Usernamen und Password in URL, z.Bsp.: username:password@$IP$.

    Blaue Iris

    Öffne den "Add Kamera" dialog wie immer

    Wähle den "MJPEG Stream" von der Liste anstatt einer spezifischen Kamera.

    Video path: /video
    Audio path: /audio.wav
    Host name: $IP$
    Port: $PORT$

    Es ist unwichtig was sie eingeben in "RTSP/video port" box.

    webcamXP

    Im webcamIP main interface, rechts klicken Sie auf video source und selektiere Network Kameras ⇨ Connect...

    Setze Camera Brand auf Android und Camera Model auf IP Webcam.

    Selektiere desired preset (MJPEG empfohlen). Klicke Next.

    Setze Hostname auf $IP$ und Port auf $PORT$.

    Klicke Ok, und sie können ihre neue Kamera benutzen!

    Fortgeschritten

    Hier ist eine Liste der IP Webcam URLs zur Nutzung/Steuerung der Kamera:

    • $URL$/video ist die MJPEG URL.
    • $URL$/shot.jpg holt das neuste Videoframe als JPG Bild.
    • $URL$/audio.wav ist der Audiostream im Wav format.
    • $URL$/audio.aac ist der Audiostream in AAC format (sofern vom Gerät unterstützt).
    • $URL$/audio.opus ist der Audiostream im Opus format.
    • $URL$/focus fokusiert die Kamera.
    • $URL$/nofocus schaltet den Fokus der Kamera ab.

    Zugriff auf IP-Webcam außerhalb lokalen WiFi (e.g. via the internet)

    Sie können ihre Kamera connecten via Internet, wenn ihr Smartphone verbunden ist via WiFi und der WiFi Router eine externe IP Addresse hat(ist meistens der Fall).

    Die Konfiguration erfordert etwas Arbeit, aber ist machbar für die Meisten.

    Wichtig: Bitte lesen sie den MAC binding guide um IP Addressenänderungen zu vermeiden.

    Via Web interface ihres Routers, sollten sie den Server Port forwarden zum Internet und Zugriff auf die Kamera bekommen via externen IP des Routers. Diese Function nennt man "Port forwarding", "Virtual servers".

    Lesen Sie die Dokumentation des Routers. Es existieren freie Datenabanken von Manuals über Port Forwarding auf http://portforward.com/. Ignorieren Sie die zahlungspflichtigen Angebote.

    Der Setup sieht so aus:

    Phone (192.168.3.198, server at 8080) ---> Router (Forwarding 192.168.3.198:8080, externe IP ist 66.77.88.99) <--- Jeder Client mit Internet Access (öffnen Sie http://66.77.88.99:8080).

    Es gibt eine einfache Möglichkeit um zu Testen ob die IP Webcam sichtbar ist im Internet. Falls die exteren IP Address nicht finden können,

    in den Einstellungen des Routers , dann gehen sie auf http://whatismyip.org.

    Um zu Prüfen ob ihr Router richtg konfiguriert ist, können Sie auch einen online port scanner benutzen, z.Bsp. this one. Tippen sie ihre externe IP in das IP Feld (im Bsp., 66.77.88.99), und der externe geforwardede Port in "list of ports" field (8080 in my example).

    Bemerken Sie, dass sie wahrscheinlich kein Zugriff auf ihr Smartphone haben via die externe Router Schnittstelle,solange sie innerhalb

    des gleichen Netzwerks des Routers sind.

    Das ist ein bekannter Bug in der Firmware des Routers. Sie müssen in einem anderem WiFi Netzwerk sein oder via 3g .

    Warum ändert sich die IP Addresse andauernd?

    Das Geräte zuständig für die IP-Vergabe ist der WiFi Router. The IP Vergabe ("lease") ist gültig für bestimme Zeit und ändert sich dann automatisch.

    Das könnte ihre IP Webcam nicht erreichbar machen mit der alten IP Addresse.

    Um das zu Verhindern, tippen Sie im Web Interface des Routers und legen Sie eine statische IP Addresse des Gerätest fest.

    Suche für Menu Punkte ähnlich wie "DHCP clients", "Static leases" oder "IP/MAC binding", und füge eine MAC Anbindung zur gewünschten IP Address hinzu.

    Die MAC address ihres Smartphones finden sie unter Settings->About Phone->Status->WiFi MAC address.
