Warum hört man den Ton 4 Sekunden später?

Das passiert wegen des Buffering vom Browser und hängt vom jeweiligem Browser ab.

Versuchen Sie diese Lösungen:

  • Benutze den Flash Renderer. Flash Renderer ist designed für reduzierte Zeitdifferenz.
  • Benutze Firefox mit Opus audio. Opus ist die next-gen low-latency low-bandwidth codec audio streaming codec und Firefox macht nur wenig Buffering.

  • Benutze VLC Media Player. Im Menu, öffne Media ⇨ Open Network Stream, tippe http://your-ip:your-port/audio.wav als URL, klicke "Show more options"

    und setze das Buffering auf 1 ms.

Tasker Ereignisse Kontrolle

Tasker ist eine Automationssoftware für Android Smartphones. Es hilft ihnen zum Beispiel, ein SMS zu senden oder die Ruftonlautstärke zu ändern, je nach GPS Positionb oder Zeitplan.

IP Webcam Pro kann sogenannte Tasker Ereignisse generieren, entweder via Webinterface oder mit einem anpassbarem Benutzerinterface.

Um den Tasker für IP Webcam Ereignisse zu konfigurieren, kreiere ein neues Profil mit dem "+" Kopf und wähle State ⇨ Plugin ⇨ IP Webcam als Trigger.

Ereignisse welche via Webinterface erstellt werden, haben den gleichen Ereignisnamen wie die Shclatflächenbeschriftung.

Bei Verwendung von HTTPS warnen?

Wenn der Host nicht über die DNS aufgerufen wird, ist eine HTTPS-Warnung normal.

Die Verwendung von HTTPS schützt vor Lauschangriffen, wenn die Netzwerke unsicher sind.

Diese Warnungen können ignoriert werden.
